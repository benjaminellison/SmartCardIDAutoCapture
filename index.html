Smart Card Disposal Logger

How to use

Hold a stack of cards in front of the webcam, back of the top card facing the lens. The detector waits for a still, sharp white card and snaps automatically. Peel the top card off — that motion re-arms the detector. Repeat.

  • Space — force capture the current frame
  • U — undo last add
  • J / K — next / previous review item
  • Enter — confirm focused review item
  • D — discard focused review item
  • Esc — close settings / collapse review

Set your OpenRouter API key in Settings, or via .env.local. Bad reads land in the review tray below; physically set those cards aside and clean up at the end.

Full setup, tuning guide, and architecture notes: github.com/benjaminellison/SmartCardIDAutoCapture
